# LyricViewX

LyricViewX is a view to show lyrics. It based on [LrcView](https://github.com/zion223/NeteaseCloudMusic-MVVM).

# What's new
- 100% Kotlin.
- It's new and beautiful.
- It's easy to use.

# How to use

```groovy
dependencies {
    implementation 'com.github.Moriafly:LyricViewX:1.0.0'
}
```
